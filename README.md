# ⚡ Assistant IA Dashboard

Une interface de dashboard moderne, minimaliste  conçue avec **Next.js 15** et **Tailwind CSS**. Ce projet inclut un système de chat IA complet, une gestion de thème (sombre/clair) et une navigation responsive.

## 🚀 Fonctionnalités

- 📱 **Design Ultra-Responsive** : Sidebar escamotable et interface adaptée aux mobiles.
- 🌓 **Mode Sombre/Clair** : Intégration fluide avec `next-themes`.
- 🤖 **Interface de Chat IA** : Expérience utilisateur inspirée des meilleurs outils d'IA.
- ⏳ **Écran de Chargement Premium** : Transition élégante lors de l'accès au dashboard.
- 🔐 **Système de Navigation** : Redirections intelligentes et page de login dédiée.

## 🛠️ Stack Technique

- **Framework** : [Next.js 15 (App Router)](https://nextjs.org/)
- **Stylisation** : [Tailwind CSS](https://tailwindcss.com/)
- **Icônes** : [Lucide React](https://lucide.dev/)
- **Polices** : [Google Fonts (Poppins)](https://fonts.google.com/)
- **Déploiement** : [Vercel](https://vercel.com/)
